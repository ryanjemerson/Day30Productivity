Laptop Setup
Follow the steps below to image your REISSUED laptop
  • Plug in power supply and attach Network cable (for laptops that require Ethernet, use the dongle specified for that device)
  • Turn on computer, press or rapidly tap the F12 key, if F12 doesn’t work, try ESC, F1,or F9. For Surface devices, hold the power and down volume buttons.
  • Select an option: Onboard NIC, PCI LAN, IPV4, and then look for a text prompt (pressing F12 again or pressing Enter). On some laptops use Fn + F12
  • When the boot manager appears, select MSIT ALL Supported Windows Client, it may take time to load
  • Select Let’s get started and follow steps

Post Installation

  • Click on Windows > Settings > Update and Security, check the online box and run the update
Laptop Setup
Follow the steps below to image your NEW laptop
  • Plug the power cord into your computer, and then plug it into an electrical socket
  • Attach a network cable to your computer, and then plug it into the wired corporate network
  • Start up your device
    • The Setup program displays a progress indicator window as it gathers information about your computer. The Setup program then displays a Let’s get started page
  • On the Let’s get started page, ensure that the appropriate keyboard layout is selected, and then select the arrow icon
    • The Setup program automatically runs through a series of deployment readiness tests and displays the results on the Deployment Readiness page. On the Deployment Readiness page, review the test results, and then select Next if there are no errors
  • If a particular test is unsuccessful (a red X appears on the page), resolve the problem, and then select the Retry Checks button
  • Select Next.
  • On Computer Details page, change the computer name (do not use the suggested computer name). This will make it easier to join a domain automatically

NOTE: If you cannot join a domain at this time because you do not have a connection to the wired corporate network, you can select Join a workgroup on the Computer Details page and proceed with installing Windows 10. If you do this, you will need to join a domain (and turn BitLocker on) at a later time, when a connection to the wired corporate network is available. Instructions for doing so are provided in the “After installing” section of this guide, next

  • Verify the information in the fields under Join a domain, enter your network password, confirm your password, and then select Next
  • On the Language Packs page, select any additional language packs you want to install, and then select Next
  • On the Locale and Time page, verify the settings, and then select Next
  • On the Administrator Password page, enter an administrator password for the local admin account, confirm it, and then select

NOTE:This is a password for the local admin account, not your network account. Be sure to make note of the password for recovery purposes.

  • On the Local Administrators page, if you do not see your domain and user name already filled in for you, add using the format domain/username
  • Select one of the following:
    • If you will be the only user on the computer > Next
    • If other users will be on this computer, enter additional users > Next
  • On the BitLocker page, select Next to enable BitLocker

NOTE:This step assumes that you are connected to the wired corporate network and were therefore able to select Join a domain on the Computer Details page in Step 6. Join the corporate network
Follow the steps below to domain join your laptop
  • Right click > System
  • Click System > Change Settings
  • Select Change
  • Enter domain name in Domain option, i.e. redmond.corp.microsoft.com
  • When complete a Welcome to Domain message will appear, follow additional steps before restarting computer.
  • Click Window > Manage > Computer Management
  • Expand Local Users and Groups > Groups > Administrators > add , enter domain\alias
  • Press Windows-L on your keyboard, on logon page > Other User, enter your corporate user account credentials, i.e. redmond\<username> and password
  • When Start menu appears, restart your computer to complete the domain join process
Enable Bitlocker
Follow steps below to set up Bitlocker
  • Start > Control Panal > BitLocker Drive Encryption
  • Turn On BitLocker
  • Encrypt drive
  • Follow the wizard
Setup your Outlook profile
Follow steps below to set up your Outlook Profile
  • Left click > Outlook
  • Click Next at Welcome Screen
  • Click Yes to add new email account > Next
  • In the Auto Account Wizard, enter your name, email address, and password
  • Click Next > Finish
Sign into Skype for Business
Sign into Skype for Business
  • Left click type Skype for Business
  • Fill in your name and email address
  • Click Yes to add new email account > Next
  • In the Auto Account Wizard, enter your name, email address, and password
  • Click Next > Finish
